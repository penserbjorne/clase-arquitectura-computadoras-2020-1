# Configuración de Pines

## Entradas

- clk           =>  Pin 17
- reset         =>  pin 101 => Se activa en bajo
- izquierda     =>  Pin 99
- derecha       =>  Pin 96

## salidas

- adelante  =>  Pin 93
- atras     =>  Pin 92
- izquierda =>  Pin 86
- derecha   =>  Pin 87
- estado0   =>  Pin 73  => bls (bit menos significativo)
- estado1   =>  Pin 75
- estado2   =>  Pin 79
- estado3   =>  Pin 74  => bms (bit más significativo)
